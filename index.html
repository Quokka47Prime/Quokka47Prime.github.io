RSync Folders

A handy file syncronization tool available soon in the Mac App Store.

Backs up and synchronizes files inside a main folder and all sub folders using the powerfull RSync script utility.

Sub-folder scripts run simultaneously for a very fast folder sync. Latest changes are synced to/from local and remote locations bi-directionally.
