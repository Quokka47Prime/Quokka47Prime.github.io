Easy RSync

A file syncronization tool available in the Mac App Store.

Backs up and synchronizes files from a folder and all it's sub-folders. Sync All feature runs sub-folder scripts simultaneously for multiprocessing speed.

Uses the UNIX RSync script to move latest changes bi-directionally from both local and remote locations.
